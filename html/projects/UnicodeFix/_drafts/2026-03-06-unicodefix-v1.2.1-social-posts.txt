UnicodeFix v1.2.1 is out.

This release is less about flashy new tricks and more about removing friction:

- a unified install path
- better Conda behavior
- improved Unicode hyphen handling, including non-breaking hyphens
- cleaner `--metrics` behavior
- more honest reporting, especially around quote-like characters
- better Python compatibility and CI coverage

In short: fewer rough edges, fewer false alarms, and a much smoother day-to-day experience.

If you've ever had text that looked fine but still broke CI, diffs, formatting, or shell workflows, this is exactly the kind of update that matters.
